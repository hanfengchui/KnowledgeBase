# 星澜云服企业知识库测试文档

## 文档用途

这是一份用于测试“企业知识库 + AI 问答助手”的样例文档。上传后可以验证文档入库、片段召回、来源展示、问答生成和无依据回答等功能。

## 公司简介

星澜云服是一家面向企业客户的数据智能服务商，主要产品是 InsightFlow 企业数据助手。InsightFlow 可以帮助销售、客服和运营团队把分散在数据库、表格、工单和知识库中的信息统一检索，并通过 AI 助手生成业务回答。

## 产品功能

InsightFlow 支持三类核心功能：

1. 企业知识库问答：把产品手册、售后政策、FAQ 和内部流程文档导入系统，员工可以直接用自然语言提问。
2. 数据看板解释：系统可以读取销售额、转化率、客诉率等指标，并生成面向业务人员的解释。
3. 工单辅助处理：客服提交问题后，系统会自动检索历史案例，推荐回复模板和处理步骤。

InsightFlow 第一版支持 Markdown、TXT 和 CSV 文件导入。PDF 和 Word 文件解析属于后续版本规划，不在第一版范围内。

## 支持的数据源

InsightFlow 支持以下数据源：

- PostgreSQL
- MySQL
- 企业内部 HTTP API
- CSV 文件
- Markdown 文档
- TXT 文本文档

如果客户需要连接 Oracle、SQL Server 或 Elasticsearch，需要由项目经理评估后走定制化集成流程。

## 退款规则

企业客户购买 InsightFlow 标准版后，可以在合同生效后 7 个自然日内申请退款。退款必须满足以下条件：

- 客户尚未完成正式生产环境部署。
- 客户没有使用超过 3 个企业账号。
- 客户提交书面退款原因。
- 销售和客户成功经理共同确认没有未完成的定制开发交付。

如果客户购买的是定制化项目包，已经完成需求确认、接口开发或私有化部署后，不支持无理由退款。若交付内容与合同约定明显不一致，可以发起专项售后评审。

## 发票规则

客户付款后可以申请增值税普通发票或增值税专用发票。开票信息必须与合同主体一致，包括公司名称、税号、开户地址、开户银行和银行账号。

发票默认在付款确认后的 5 个工作日内开具。如果客户需要提前开票，需要销售主管审批，并由财务确认风险。

## 售后 SLA

InsightFlow 标准版售后支持时间为工作日 9:00 到 18:00。

不同故障等级的响应时间如下：

- P1 级故障：核心问答服务不可用，30 分钟内响应。
- P2 级故障：部分数据源无法同步，2 小时内响应。
- P3 级问题：配置咨询、使用咨询或普通优化建议，1 个工作日内响应。

如果客户购买高级支持包，可以获得 7x24 小时紧急响应通道。高级支持包不包含免费定制开发。

## 权限与安全

InsightFlow 支持基于角色的权限控制。系统内置三类角色：

- 管理员：可以管理数据源、用户、知识库和系统配置。
- 业务用户：可以提问、查看授权知识库和使用业务工具。
- 只读访客：只能查看公开知识库中的回答，不能上传文档。

所有上传的知识库文档都会记录上传人、上传时间、文件哈希和文档来源。系统默认不会把企业原始文档用于模型训练。

## 部署要求

标准私有化部署至少需要以下资源：

- 4 核 CPU
- 16GB 内存
- 200GB 磁盘
- PostgreSQL 15 或更高版本
- 一个可访问大模型 API 的网络出口

如果客户需要向量检索能力，数据库需要安装 pgvector 扩展。向量维度必须与使用的 embedding 模型保持一致。

## 第一版不包含的内容

本测试文档没有提供以下信息：

- 加班餐补标准
- 年终奖规则
- 员工绩效等级
- 公司班车路线
- VPN 账号申请流程

如果用户提问这些问题，AI 助手应该说明知识库未提供足够依据，而不是编造答案。
